### Команды:
- 📊 Получить помощь - "**.help**"
- 📰 Получить статистику по capture - "**.get server_cpt**"
- 📰 Получить статистику по bizwar - "**.get server_biz**"

---

#### Вместо "server" необходимо записать соответствующий сервер.
